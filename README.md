## irl-time

**How long does it actually take? 💭**

Inspired by how [Christi Newrutzen](https://www.instagram.com/reel/DSLNEoVkmEW/?utm_source=ig_web_copy_link&igsh=NTc4MTIwNjQ2YQ==) set out to time exactly how long "overwhelming" tasks take to call out the procrastinators (guilty!);

I built a timer for my desktop to log exactly how long my boring work takes (sigh, release notes) so that I plan my day with irl time instead of "ugh-this-will-take-forever" time
